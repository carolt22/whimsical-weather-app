# Feature Breakdown

## Dynamic Themes 
Themes change based on:
- weather condition (rain, clouds, snow, etc.)
- time of day (day/night)

Unique backgrounds coupled with animation creates a more immersive user experience. 

## Stargazing rating system
Rating is computed using:
- cloud coverage
- weather conditions
- time of day

Adjusted further to account for moon phase brightness.

## Star Field Rendering 
Stars are dynamically generated using JavaScript
- number of stars dependent on visibility rating
- twinkle animation applied conditionally

## Expandable Card System
Cards make use of: 
- event listeners
- DOM state toggling
- conditional rendering

Allows for progressive disclosure of information.
